# 🚀 C++ Real-Time Rendering Engine (DirectX)

C++ real-time rendering engine using DirectX, featuring custom pipeline, FXAA, particles and shader hot-reloading.

Custom 3D engine developed in C++ using DirectX.

## ✨ Features

- Real-time rendering pipeline
- Scene management system
- FXAA (anti-aliasing)
- Particle system
- Shader hot-reloading
- Debug console (Quake-like)
- Physics integration (Newton Dynamics)
- Custom math library (vectors, matrices, planes)

---

## 🧱 Architecture

The engine is structured into multiple subsystems:

- **Renderer**: Handles draw calls, shaders, and GPU communication
- **Scene system**: Manages objects, transforms, and visibility
- **Renderable abstraction**: Encapsulates buffers and rendering data
- **Math module**: Custom implementation for transformations and geometry
- **Tools**: Debug console, shader hot reload
